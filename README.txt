Snickers Cup App - samlet version

Funktioner:
- Resultater og live stilling.
- Spilleropslag.
- Kampplan.
- Excel-upload af spillerliste.
- Formatvalg mellem standard og 8 hold á 5 spillere.
- Delbart link med den spillerliste, du har indsat.
- Firebase live setup, hvis resultater og spillerliste skal synkroniseres på tværs af enheder.

Brug:
1. Upload filerne til GitHub Pages.
2. Åbn appen i browseren.
3. Gå til Spillere for at uploade Excel.
4. Gå til Format for at vælge turneringsformat.
5. Brug "Lav delbart link med spillerliste" hvis andre skal åbne appen med samme spillerliste.
6. Brug Live setup hvis resultater skal deles live mellem flere enheder.

Bemærk:
- Delbart link deler spillerliste og format, men ikke resultater.
- Live resultater kræver Firebase opsætning.
- Ved formatskift nulstilles resultaterne.
